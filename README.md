# Der 5x5-Code

Ein kleines GitHub-Pages-Projekt für den Unterricht zu geheimen Schriften.

## Funktionen

- Botschaften verschlüsseln und entschlüsseln
- Quadrate von 5x5 bis 10x10 wählbar
- automatische Füllbuchstaben
- Ausgabe des Geheimtexts in 5er-Gruppen
- Visualisierung des verwendeten Quadrats

## Verwendung auf GitHub Pages

1. Repository erstellen
2. Alle Dateien hochladen
3. Unter **Settings → Pages** GitHub Pages aktivieren
4. `index.html` öffnet die App direkt im Browser

## Regel

Beim Verschlüsseln wird der Klartext senkrecht in das Quadrat geschrieben. Danach wird der Geheimtext waagerecht gelesen.
Beim Entschlüsseln wird der Geheimtext waagerecht eingetragen und senkrecht gelesen.
